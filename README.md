# RFToolkit

## Portable Open Source SDR Research and Mobile RF Experimentation Framework

RFToolkit is an open-source modular radio frequency (RF) experimentation and analysis framework designed to simplify software-defined radio research and improve SDR accessibility. The project focuses on enabling portable SDR workflows, protocol research, signal analysis, and wireless communication experimentation using HackRF devices.

The long-term vision of RFToolkit is to reduce reliance on traditional SDR host setups such as laptops or dedicated expansion hardware by enabling mobile-device-driven SDR experimentation environments.

## Mobile SDR Host Development

A major focus of this project is enabling software-defined radio experimentation directly from Android devices.

Current development includes:

- Custom Android kernel development enabling SDR hardware compatibility
- OTG-based HackRF connectivity support
- Rooted Android SDR integration workflows
- SDR operation through mobile terminal environments
- Optimization for Kali NetHunter SDR workflows

Development is currently being tested on:

- Motorola Edge 30 Fusion (Codename: tundra)
- Android 16 based custom kernel with SDR compatibility modifications

This work allows mobile devices to function as fully portable SDR research platforms, significantly lowering hardware barriers for RF experimentation.
